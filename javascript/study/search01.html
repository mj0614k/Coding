자바스크립트 : 검색 효과

자바스크립트의 다양한 효과들 중, 해당하는 문자열을 찾을 수 있는 검색 효과입니다. HTML의 알파벳, CSS의 알파벳, JAVASCRIPT의 메서드를 검색할 수 있는 웹 사이트를 작업해 보도록 하겠습니다.

HTML 소스

HTML로 뼈대를 먼저 잡아주었습니다. 헤더 영역에는 내비게이션 메뉴를, 메인 영역에는 검색창, 하위 목록들을 작성해 주었습니다. 내비게이션 메뉴에서 HTML, CSS, JAVASCRIPT를 클릭하면 각각의 페이지로 이동할 수 있습니다. 아래 소스는 메인 영역의 HTML 소스로, 나머지 영역의 소스는 하단의 '원본 소스 보기' 버튼을 클릭하면 확인할 수 있습니다.

        <div class="search__box">
            <label for="search">검색하기</label> 
            <input type="text" id="search" placeholder="HTML 태그를 입력해 주세요">
        </div>
        <div class="search__list">
            <div class="html">
                <ul>
                    <li data-name="a"><strong>a</strong> : 다른 페이지 이동을 설정합니다.</li>
                    <li data-name="abbr"><strong>abbr</strong> : 줄임말이나 머리글자를 표현하는 데 사용합니다.</li>
                    <li data-name="acronym"><strong>acronym</strong> : 줄임말이나 머리글자를 표현하는 데 사용합니다. 현재는 사용하지 않습니다.</li>

                    <li data-name="applet"><strong>applet</strong> : 	자바 애플릿을 보이게 하는 곳에 쓰입니다. 현재는 사용하지 않습니다.</li>

JAVASCRIPT 소스

대망의 JAVASCIPRT 소스입니다. 검색 영역과 목록 리스트 선택자를 먼저 만들어 준 뒤, addEventListener로 검색 영역에 keyup 이벤트를 추가해 주었습니다. 사용자가 입력한 키워드와 하위 목록들의 값을 비교하여, indexOf의 값이 0이면 리스트를 보여주고, 값이 다르면 리스트를 숨기는 방식으로 작업해 주었습니다. 

// 선택자
const searchBox = document.querySelector(".search__box input")  // 검색 영역
const searchList = document.querySelectorAll(".search__list ul li")  // 목록 리스트

searchBox.addEventListener("keyup", () => {  // 한 글자만 입력해도 검색 가능
    const searchWord = searchBox.value;  // 사용자가 입력한 키워드
    
    searchList.forEach((el)=>{
        const cssName = el.dataset.name;
        if(cssName.indexOf(searchWord)){
            el.classList.add("hide");
        } else {
            //"abbr".indexOf("abbr")
            el.classList.remove("hide");  // indexOf의 값이 0이 되는 값만 hide 클래스 삭제
        }
    });
});
