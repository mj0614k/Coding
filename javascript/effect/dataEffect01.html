1.4 for문을 이용해서 1부터 100까지의 합 출력하기

let sum = 0;
for(let i=1; i<=100; i++){
    sum = sum + i;  // 첫번째 방법
    sum += i;          // 두번째 방법
}
document.write(sum);
결과보기
5050
